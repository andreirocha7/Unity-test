# Unity-test

Tutorial de Automação de Testes em C com Unity

Este repositório apresenta um tutorial prático de configuração e uso do framework de testes unitários Unity para programas em linguagem C.
O objetivo é demonstrar, passo a passo, como automatizar a execução de testes unitários, desde a organização do projeto até a geração do output dos testes.

O Unity é um framework leve, amplamente utilizado em projetos C, especialmente em sistemas embarcados e ambientes educacionais, por não exigir instalação complexa e funcionar apenas com arquivos .c e .h.

A estrutura de diretórios do projeto é apresentada a seguir:

Atividade/
 ├── src/              (código de produção)
 ├── test/             (código de teste)
 ├── unity/            (Unity framework)
 ├── build/            (arquivos compilados)
 └── Makefile
